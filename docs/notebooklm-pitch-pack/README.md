# NotebookLM Pitch Pack - Crucible

This folder contains curated source files for generating the final Crucible pitch deck in NotebookLM.

## How To Use

1. Upload every `.md` file in this folder to NotebookLM.
2. Open `00_notebooklm_instructions.md` first.
3. Use the prompt in the "Best Prompt To Use In NotebookLM" section.
4. Ask NotebookLM to generate a polished 12-slide Canva-ready pitch deck.
5. Import the generated content into Canva.
6. Polish visuals, screenshots, icons, and spacing in Canva.

## What This Pack Is For

This pack is designed to make NotebookLM produce a strong final deck, not just summarize the codebase.

It gives NotebookLM:

- The pitch objective
- The audience
- The deck structure
- The product story
- The technical context
- The demo narrative
- The 0G positioning
- The claims to use and avoid
- The Canva visual direction

## Important Guardrail

Do not let NotebookLM invent deployment addresses, mainnet claims, or verification guarantees.

Use official HackQuest event context and the curated framing in these files.
